
# ATM Project Folder

Ye folder ATM Management System se related files contain karta hai.
Is project ka purpose Python aur CSV file ka use samajhna hai.

## Files Included

### 1. atm2.py
- ATM system ka main Python program
- User se input leta hai (withdraw, balance check, etc.)
- CSV file ke data par operations perform karta hai

### 2. atmdata.csv
- ATM users ka data store karta hai
- Account number, balance, transaction details jaise records hote hain
- Python program isi file ko read/write karta hai

## How to Run
1. Python install hona chahiye
2. Terminal / CMD me command run kare:
   ```bash
   python atm2.py
